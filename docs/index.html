ROMBOR

Папка docs/ готова для GitHub Pages (Deploy from a branch).

  1. Соберите проект локально с базовым префиксом: BASE_PATH=/REPO_NAME/ pnpm run build:client
  2. Скопируйте содержимое dist/spa/ в docs/ с заменой этого файла
  3. Скопируйте docs/index.html в docs/404.html (SPA fallback)

После пуша страница будет доступна по адресу: https://USERNAME.github.io/REPO_NAME/
